Java 反射（Reflection）是一个强大的特性，它允许程序在运行时查询、访问和修改类、接口、字段和方法的信息。反射提供了一种动态地操作类的能力，这在很多框架和库中被广泛使用，例如Spring框架的依赖注入。

主要的API

Java 的反射 API 提供了一系列的类和接口来操作 Class 对象。主要的类包括：

java.lang.Class：表示类的对象。提供了方法来获取类的字段、方法、构造函数等。
java.lang.reflect.Field：表示类的字段（属性）。提供了访问和修改字段的能力。
java.lang.reflect.Method：表示类的方法。提供了调用方法的能力。
java.lang.reflect.Constructor：表示类的构造函数。提供了创建对象的能力。

# 工作流程

1、获取 Class 对象：首先获取目标类的 Class 对象。

2、获取成员信息：通过 Class 对象，可以获取类的字段、方法、构造函数等信息。

3、操作成员：通过反射 API 可以读取和修改字段的值、调用方法以及创建对象。

  1. 获取 Class 对象

通常有三种方式获取Class对象

①通过字面量

Class<?> clazz = String.class;

②通过对象实例

String str = "Hello";
Class<?> clazz = str.getClass();

③通过Class.forName()方法

Class<?> clazz = Class.forName("java.lang.String");

2、创建对象

可以使用反射动态创建对象

Class<?> user = Class.forName("User");
Constructor<?> constructor = user.getDeclaredConstructor(String.class,int.class);
Object obj = constructor.newInstance("张三",30);
System.out.println("创建的对象: " + obj);

3、访问字段

可以通过反射来访问和修改字段

Class<?> clazz = Class.forName("User");
Object instance = clazz.getDeclaredConstructor().newInstance();
Field field = clazz.getDeclaredField("name");
field.set(instance, "张三");
Field nameField = clazz.getDeclaredField("name");
nameField.setAccessible(true); // 如果是 private 字段，必须开启访问权限
Object value = nameField.get(instance);
System.out.println("获取到的字段值: " + value);

4、调用方法

//我们可以通过getDeclaredMethods来获取所有的方法
Class<?> clazz = Class.forName("User");
Method[] declaredMethods = clazz.getDeclaredMethods();
for(Method F: declaredMethods ){
    System.out.println(F);
}

//随后我们可以调用方法
//(静态方法写法)
Class<?> clazz = Class.forName("User");
Method sayhello = clazz.getDeclaredMethod("sayhello");
sayhello.setAccessible(true);
sayhello.invoke(null);


//(非静态方法写法)
Class<?> clazz = Class.forName("User");
Object userInstance = clazz.getDeclaredConstructor().newInstance();
Method saybye = clazz.getDeclaredMethod("saybye",String.class);
saybye.setAccessible(true);
saybye.invoke(userInstance,"张三");


//(无参方法写法)
Class<?> clazz = Class.forName("User");
Object userInstance = clazz.getDeclaredConstructor().newInstance();
Method saybye = clazz.getDeclaredMethod("saybye");
saybye.setAccessible(true);
saybye.invoke(userInstance);

5、获取构造函数

可以获取和调用构造函数

Class<?> clazz = Class.forName("User");
Constructor<?> constructor = clazz.getConstructor(String.class,int.class);
Object o = constructor.newInstance("张三", 30);
System.out.println(o);
