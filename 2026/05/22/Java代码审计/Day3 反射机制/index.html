# 1. 什么是反射

Java 反射（Reflection）是 Java 非常重要的动态特性，通过使用反射我们不仅可以获取到任何类的成员方法（Methods）、成员变量（Fields）、构造方法（Constructors）等信息，还可以动态创建 Java 类实例、调用任意的类方法、修改任意的类成员变量值等。Java 反射机制是 Java 语言的动态性的重要体现，也是 Java 的各种框架底层实现的灵魂。

反射是 Java 安全的核心中的核心，90% 的利用链都依赖反射。

涉及关键字 含义
declared 声明的
invoke 调用
newInstance 新实例

# 2. 获取Class对象

介绍三种常用的获取 Class 对象的方法：

方式 代码 适用场景
类名.class Runtime.class 编译时已知类名
对象.getClass() obj.getClass() 已有实例对象
Class.forName() Class.forName("全限定名") 运行时动态加载（安全研究最常用） 
// 方式1：类名.class — 编译时就知道类名
Class<?> clazz1 = Runtime.class;

// 方式2：对象.getClass() — 手头已经有实例对象
Runtime rt = Runtime.getRuntime();
Class<?> clazz2 = rt.getClass();

// 方式3：Class.forName("全限定类名") — 运行时才知道类名（安全里最常用）
Class<?> clazz3 = Class.forName("java.lang.Runtime");

# 3. 获取类的成员方法

方法 说明
aClass.getDeclaredMethod("方法名", 参数类型...) 获取该类中指定的方法，包括private
aClass.getDeclaredMethods() 获取该类中声明的所有方法，包括private（不含继承）
aClass.getMethod("方法名", 参数类型...) 获取该类以及父类中指定的public方法
aClass.getMethods() 获取该类以及父类中所有的public方法（含继承）

示例代码：

package com.syz;

public class reDemo {
    public String name;

    public void hello(){
        System.out.println("这是公共方法");
    }
    public static void hello1(String name){
        System.out.println("你好:" + name);
    }
    private void hello2(){
        System.out.println("这是私有方法");
    }
    private static void hello3(String name){
        System.out.println("你好:" + name);
    }
    void hello4(){
        System.out.println("这是默认方法");
    }
    protected void hello5(){
        System.out.println("这是受保护的方法");
    }
}

获取方式

Class<?> aClass = Class.forName("com.syz.reDemo");

// 获取该类全部的成员方法，包括 private
Method[] declaredMethods = aClass.getDeclaredMethods();
for (Method declaredMethod : declaredMethods) {
    System.out.println(declaredMethod.getName());
}
// [返回  hello4 hello3 hello2 hello1 hello hello5]

// 获取该类及父类所有的成员方法，不包括 private
Method[] methods = aClass.getMethods();
for (Method method : methods) {
    System.out.println(method.getName());
}
// [返回  hello1 hello wait wait wait equals toString hashCode getClass notify notifyAll]

成员方法有两个维度的区别：

  1. 静态 / 实例 — 调用区别：静态方法无需实例对象，实例方法需要先创建对象
  2. 公共(public) / 受保护(protected) / 默认(default) / 私有(private) — 调用区别：反射可以绕过访问控制，非 public 方法（private、default、protected）通过 setAccessible(true) 即可访问

调用方式

Class<?> aClass = Class.forName("com.syz.reDemo");

// 1. 公共实例方法
Method method = aClass.getDeclaredMethod("hello");
Object o = aClass.getDeclaredConstructor().newInstance();
method.invoke(o);

// 2. 公共静态方法
Method method1 = aClass.getDeclaredMethod("hello1", String.class);
method1.invoke(null, "张三");

// 3. 私有实例方法 — 需要 setAccessible(true)
Method method2 = aClass.getDeclaredMethod("hello2");
method2.setAccessible(true);
Object o1 = aClass.getDeclaredConstructor().newInstance();
method2.invoke(o1);

// 4. 私有静态方法 — 需要 setAccessible(true)
Method method3 = aClass.getDeclaredMethod("hello3", String.class);
method3.setAccessible(true);
method3.invoke(null, "张三");

// 5. 默认方法
Method hello4 = aClass.getDeclaredMethod("hello4");
Object o3 = aClass.getDeclaredConstructor().newInstance();
hello4.invoke(o3);

// 6. 受保护的方法
Method hello5 = aClass.getDeclaredMethod("hello5");
Object o4 = aClass.getDeclaredConstructor().newInstance();
hello5.invoke(o4);

总结：

  • 实例方法需要先创建实例对象：aClass.getDeclaredConstructor().newInstance()
  • 私有方法需要设置特权：method.setAccessible(true)

# 4. 获取和使用类的成员变量

方法 说明
aClass.getDeclaredField() 获取该类中指定的变量，包括private
aClass.getDeclaredFields() 获取该类中所有的变量，包括private
aClass.getField() 获取该类以及父类中指定的变量,不包括private
aClass.getFields() 获取该类以及父类中所有的变量,不包括private

示例代码

public class reDemo{
    public String name = "张一";
    protected String name1 = "张二";
    String name2 = "张三";
    private String name3 = "张四";
    private static String name4 = "张五";
}

获取方式

Class<?> aClass = Class.forName("com.syz.reDemo");

// 获取该类及父类的成员变量，不包括 private
Field[] fields = aClass.getFields();
for (Field field : fields) {
    System.out.println(field.getName());
}
// [返回  name]

// 获取该类的全部成员变量，包括 private
Field[] declaredFields = aClass.getDeclaredFields();
for (Field field : declaredFields) {
    System.out.println(field.getName());
}
// [返回  name  name1  name2  name3  name4]

成员变量有两个维度的区别：

  1. 静态 / 实例 — 调用区别：静态变量无需实例对象（传 null），实例变量需要先创建对象
  2. 公共(public) / 受保护(protected) / 默认(default) / 私有(private) — 调用区别：反射可以绕过访问控制，非 public 变量（private、default、protected）通过 setAccessible(true) 即可访问

调用方式

当变量是实例时，我们需要先创建实例对象，才能调用成员变量

读取变量值

// 公共实例属性
Field name = aClass.getDeclaredField("name");
Object o = aClass.getDeclaredConstructor().newInstance();
System.out.println(name.get(o));  // [返回 张一]

// 受保护的实例属性
Field name1 = aClass.getDeclaredField("name1");
Object o1 = aClass.getDeclaredConstructor().newInstance();
System.out.println(name1.get(o1));  // [返回 张二]

// 默认实例属性
Field name2 = aClass.getDeclaredField("name2");
Object o2 = aClass.getDeclaredConstructor().newInstance();
System.out.println(name2.get(o2));  // [返回 张三]

// 私有实例属性 — 需要 setAccessible(true)
Field name3 = aClass.getDeclaredField("name3");
name3.setAccessible(true);
Object o3 = aClass.getDeclaredConstructor().newInstance();
System.out.println(name3.get(o3));  // [返回 张四]

// 私有静态属性 — 传 null 即可
Field name4 = aClass.getDeclaredField("name4");
name4.setAccessible(true);
System.out.println(name4.get(null));  // [返回 张五]

修改变量值

// 修改私有静态属性
Field name4 = aClass.getDeclaredField("name4");
name4.setAccessible(true);
name4.set(null, "王五");
System.out.println(name4.get(null));  // [返回 王五]

// 修改私有实例属性
Field name3 = aClass.getDeclaredField("name3");
name3.setAccessible(true);
Object o = aClass.getDeclaredConstructor().newInstance();
name3.set(o, "王五");
System.out.println(name3.get(o));  // [返回 王五]

// 修改公共实例属性
Field name = aClass.getDeclaredField("name");
Object o1 = aClass.getDeclaredConstructor().newInstance();
name.set(o1, "王五");
System.out.println(name.get(o1));  // [返回 王五]

# 5. 获取类的构造器

方法 说明
aClass.getDeclaredConstructor(参数类型...) 获取该类中指定的构造方法，包括private
aClass.getDeclaredConstructors() 获取该类中所有的构造方法，包括private
aClass.getConstructor(参数类型...) 获取该类中指定的public构造方法
aClass.getConstructors() 获取该类中所有的public构造方法

示例代码

public class reDemo{
    public String name;

    public reDemo(){
        this.name = "无参构造";
    }

    private reDemo(String name){
        this.name = name;
    }

    protected reDemo(String name, int age){
        this.name = name;
    }
}

获取方式

Class<?> aClass = Class.forName("com.syz.reDemo");

// 获取所有 public 构造方法
Constructor<?>[] constructors = aClass.getConstructors();
for (Constructor<?> c : constructors) {
    System.out.println(c);
}

// 获取所有构造方法（包括 private）
Constructor<?>[] declaredConstructors = aClass.getDeclaredConstructors();
for (Constructor<?> c : declaredConstructors) {
    System.out.println(c);
}

调用方式

// 获取无参 public 构造方法并创建实例
Constructor<?> constructor = aClass.getConstructor();
Object o = constructor.newInstance();
System.out.println(o);

// 获取私有构造方法并创建实例 — 需要 setAccessible(true)
Constructor<?> constructor1 = aClass.getDeclaredConstructor(String.class);
constructor1.setAccessible(true);
Object o1 = constructor1.newInstance("张三");
System.out.println(o1);

总结：

  • 私有构造需要 setAccessible(true) 才能访问
  • aClass.getDeclaredConstructor().newInstance() 本质是先获取无参构造器再调用 newInstance()，是创建实例的快捷方式
