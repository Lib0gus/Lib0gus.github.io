# 配置 Java 开发环境

# 1. 安装 JDK

  • JDK 8 — Java 安全研究最重要版本，大量漏洞利用链基于它
  • JDK 11 — LTS，很多企业在用
  • JDK 17 — 新 LTS，高版本安全特性
  • 配置环境变量 JAVA_HOME，学会在 IDEA 中切换 JDK 版本

为什么需要多个版本？

不同版本 JDK 的安全限制不同。很多反序列化漏洞在 JDK 8 上可以直接利用，但高版本 JDK 就需要绕过。审计时也需要判断目标用的是哪个版本。

# 2. 安装 IDEA

下载安装 IntelliJ IDEA（社区版或旗舰版均可）。

# 3. 安装 Maven

  • 配置 settings.xml，设置阿里云镜像加快依赖下载
  • 了解 pom.xml 的基本结构

# 4. 创建第一个项目

  • 用 IDEA 创建一个 Maven 项目
  • 写一个 HelloWorld，确认能跑通

# 配置 Maven

Maven 是一个项目管理与构建自动化工具，主要用于 Java 项目。核心功能包括：

  • 项目构建（编译、测试、打包、部署）
  • 依赖管理（自动下载和管理第三方库）
  • 标准化项目结构（约定优于配置）
  • 插件扩展（支持自定义构建流程）

# 下载

前往 Maven 官网 下载最新版本。
